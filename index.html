Data Backup & Restore

Backup all student data to a file that can be restored in any browser

How to transfer data between browsers:

  1. In the original browser: Click "Backup Data" to download a backup file
  2. In the new browser: Click "Restore Data" and select the backup file
  3. All student data will be transferred to the new browser
